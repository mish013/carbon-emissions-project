This project analyses Carbon Emissions between different countries using Python.
It also compares Emission trends overtime.
The dataset contains country-level emissions data with yearly values.

# Tools used 
- Python
- Matplotlib
- Pandas

# Columns used:
- country_name
- year
- value (emissions)

## What I Did
- Loaded and cleaned the dataset using pandas
- Filtered data for specific countries (UK, USA, Australia)
- Sorted data by year
- Plotted emissions trends using matplotlib
- Compared multiple countries on one graph
